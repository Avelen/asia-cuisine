# Asia Cuisine - Premium Asian Cafe

A premium Asian cafe website built with React and TypeScript, inspired by the elegant design of `mama-thai.pl`.

## Features
- **Home**: Hero banner and featured specials.
- **Menu**: Categorized list of authentic Asian dishes.
- **Reservation**: Online table booking form.
- **Contact**: Address, phone, hours, and map.
- **Responsive**: Fully optimized for mobile and desktop.
- **CI/CD**: Automatic deployment to GitHub Pages via GitHub Actions.

## Tech Stack
- React 19
- TypeScript
- Vite
- React Router 7
- Lucide React (Icons)
- GitHub Actions

## Development
```bash
npm install
npm run dev
```

## Deployment
The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.
